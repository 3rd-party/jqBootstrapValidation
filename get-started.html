Installation

  1. Download jqBootstrapValidation and include the script tags on your page: 
    <script src="//cdnjs.cloudflare.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
<!-- or use local jquery -->

<script src="jqBootstrapValidation.js"></script>
  2. Apply the jqBootstrapValidation plugin to the elements you want validation applied to 
    <script>
  $(function () { $("input,select,textarea").not("[type=submit]").jqBootstrapValidation(); } );
</script>
  3. Set up your validators and error messages (see below).

Usage

jqBootstrapValidation uses the HTML5 validators like 'required' automatically, but the default error messages are pretty uninspiring, so you'll likely want to change them.

Heres how to set the error message for a required field: 

<input 
    type="text" 
    name="username" 
Â    required 
    data-validation-required-message="Please enter your username"
>

Thats a pretty long attribute there, so lets break it down:

  • data is the prefix for data attributes to stop them interfering with other HTML attributes like 'class' or 'type'
  • validation is jqBootstrapValidation's prefix, to show we're talking about validation
  • required is the name of the validator
  • message is the name of a variable to pass to the validator, in this case the error message

For HTML5 built-ins, the 'name' of the validator is the same as the HTML5 attribute.

For the extra validators, you choose the name yourself.
