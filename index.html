Installation

  1. Download jqBootstrapValidation and put it on your server.
  2. Include the script tags on your page: 
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script> <!-- if you don't have jquery -->
<script src="/js/jqBootstrapValidation.js"></script>
  3. Apply the jqBootstrapValidation plugin to the elements you want validation applied to 
    <script>
  $(function () { $("input,select,textarea").not("[type=submit]").bootstrapValidation(); } );
</script>
