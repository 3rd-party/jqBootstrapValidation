Email address

  • Code
Attribute Value Req? Default

Number

Ensures a field contains a valid number

Note if you don't specify the 'step' parameter, the browser may assume you want integers (whole numbers).

Note some browsers will silently throw away non-numeric values without triggering validation

  • Live
  • Code
Attribute Value Req? Default
type number
step (smallest increment) 1
data-validation-number-message (your failure message) Must be a number
